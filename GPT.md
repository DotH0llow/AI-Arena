# Project Working Instructions

Whenever modifying this project:

1. Always return the ENTIRE updated project as a ZIP.
2. Apply requested changes directly to the project files.
3. Include a short CHANGELOG.md describing what changed.
4. Preserve previous completed work unless explicitly asked to remove it.
5. If browser limitations prevent a feature (e.g. autoplay audio), implement the best possible workaround and document it.
6. If additional assets are provided, place them in appropriate folders and update references.
7. At the end of every response, list the next logical tasks that can be implemented.
